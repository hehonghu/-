# 我对信息素养实践课程的认识  
信息素养是一门十分重要的课程，尤其是对我们这些学软件工程的人尤其如此，是学习与计算机有关知识的人的必备素养
## 信息素养实践课程的主要内容  
### 1. 计算机的合理使用方法  
如怎么关闭不需要的服务，如何查看**IP及Mac地址**，快捷键的使用,如何进行个性化设置
### 2. dos命令  
通过运行输入*cmd*进入dos命令输入界面，他提供一种不通过资源管理器直接管理磁盘文件的方法
### 3. makdown语法  
学习makdown语法可以让我们写的文档，博客等更美观，其支持超链接如[这是百度网站](www.baidu.com),
也支持插入代码块如
|第一章|第二章|第三章|
|:-:|:-:|:-:|
|计算机|数据|编程|
还可以支持列表如
1. 计算机
2. 文本
3. 扩展名
清晰明了，更容易得到上级领导的青睐
